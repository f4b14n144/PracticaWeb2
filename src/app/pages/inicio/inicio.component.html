Bienvenido a la Plataforma de Gestión de Cursos

Esta es una práctica enfocada en el desarrollo de una página web interactiva enfocada en la gestión de cursos con Angular, con funcionalidades de registro de cursos, visualización de información detallada de cada curso, que incluya animaciones y manipulación dinámica del DOM mediante CSS y JavaScript.

Funcionalidades Principales

  • Agregar nuevos cursos mediante un formulario interactivo.
  • Visualizar la lista de cursos disponibles con detalles importantes como nombre, instructor, y duración.
  • Eliminar cursos existentes y mantener actualizada la lista.
  • Ver detalles adicionales de cada curso sin necesidad de recargar la página.
  • Almacenar los datos de manera local utilizando el almacenamiento local del navegador (localStorage).
